## 本工具介绍

* 本工具主要打开和解析 UABEA 解包出来的JSON文件，代码中做了一些针对游戏的特殊处理，主要是针对一些文本的特殊处理，或类XML结构的支持等。
* 创建Python虚拟环境，安装依赖包，配置.env 文件，然后执行python main.py即可打开工具网页。
